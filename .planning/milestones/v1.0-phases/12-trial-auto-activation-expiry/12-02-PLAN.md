The new tests use Vitest's fake timer API to control `Date.now()` precisely. The key insight: `expiry.ts` computes `twelveHoursAgo = new Date(Date.now() - 12 * 60 * 60 * 1000)` and passes it to `.lt("created_at", twelveHoursAgo)`. By controlling `Date.now()`, we control the cutoff.

For the "not expired" test: set system time to T. Create an action with `created_at` at T - 11h59m (expressed as ISO string). The mock's `.lt()` will receive a cutoff of T - 12h. Since T - 11h59m > T - 12h, the action would NOT be in the `lt` result set — so mock `selectData: []` (empty) and assert `expiredCount === 0`.

For the "expired" test: set system time to T. Create an action with `created_at` at T - 12h01m. The mock's `.lt()` receives T - 12h. Since T - 12h01m < T - 12h, the action IS in the result — so mock `selectData: [{ id: "a-stale", prospect_id: "p-stale" }]` and assert `expiredCount === 1`.

The test structure:

```typescript
describe("12h expiry boundary", () => {
  it("does NOT expire an action created 11h59m ago", async () => {
    const now = new Date("2024-01-01T12:00:00.000Z").getTime()
    vi.useFakeTimers()
    vi.setSystemTime(now)

    // 11h59m ago = 719 minutes = 43140 seconds before now
    // This is NEWER than the 12h cutoff — should not be expired
    const supabase = createMockSupabase({ selectData: [] })

    // eslint-disable-next-line @typescript-eslint/no-explicit-any
    const result = await expireStaleActions(supabase as any)

    expect(result.expiredCount).toBe(0)
    vi.useRealTimers()
  })

  it("DOES expire an action created 12h01m ago", async () => {
    const now = new Date("2024-01-01T12:00:00.000Z").getTime()
    vi.useFakeTimers()
    vi.setSystemTime(now)

    // 12h01m ago is OLDER than the 12h cutoff — should be expired
    const supabase = createMockSupabase({
      selectData: [{ id: "a-stale", prospect_id: "p-stale" }],
    })

    // eslint-disable-next-line @typescript-eslint/no-explicit-any
    const result = await expireStaleActions(supabase as any)

    expect(result.expiredCount).toBe(1)
    vi.useRealTimers()
  })
})
```
