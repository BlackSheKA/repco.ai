# Sketch Manifest

## Design Direction

Dark, warm-zinc UI with indigo primary (matching radix-nova preset used in the real app). Sentence-case copy, no eyebrow pills, monospace for prices. Audience is mixed (indie hackers + freelancers + agencies + coaches) so language stays platform-neutral and avoids "B2B SaaS founder" framing. Pricing page must surface **value** (DMs you can send) without exposing post-purchase **burn math** (banned UI pattern per `credit_ui_no_burn_math` memory).

## Reference Points

- repco.ai current landing (dark, sentence case, no PRICING eyebrow pill)
- Linear / Vercel pricing pages — clean comparison tables with quiet hover
- Tailwind UI pricing components — for slider + table interplay reference
