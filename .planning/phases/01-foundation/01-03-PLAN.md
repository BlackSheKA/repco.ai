1. **Auth server actions (src/features/auth/actions/auth-actions.ts):**
   ```typescript
   "use server";
   ```
   
   Create three exported async functions:

   a. `signInWithEmail(formData: FormData)` -- extracts email string from formData.get("email"), calls `supabase.auth.signInWithOtp({ email, options: { emailRedirectTo: \`${process.env.NEXT_PUBLIC_SITE_URL}/auth/callback\` } })`. Returns `{ error: string }` on failure, `{ success: true }` on success.

   b. `signInWithGoogle()` -- calls `supabase.auth.signInWithOAuth({ provider: "google", options: { redirectTo: \`${process.env.NEXT_PUBLIC_SITE_URL}/auth/callback\` } })`. On success, calls `redirect(data.url)`. On error, returns `{ error: string }`.

   c. `signOut()` -- calls `supabase.auth.signOut()`, then `redirect("/login")`.

   All three create the Supabase client via: `const supabase = await createClient();` importing from `@/lib/supabase/server`.

2. **Auth callback route (src/app/auth/callback/route.ts):**
   Export an async `GET` handler. Extract `code` from searchParams and `next` (default "/"). If code exists, call `supabase.auth.exchangeCodeForSession(code)`. On success redirect to `${origin}${next}`. On error redirect to `${origin}/login?error=auth_callback_failed`.

3. **Root middleware (src/middleware.ts):**
   Import `updateSession` from `@/lib/supabase/middleware`. In the middleware function:
   - Call `const { response, supabase } = await updateSession(request);`
   - Get user: `const { data: { user } } = await supabase.auth.getUser();`
   - If NO user AND path is NOT `/login`, `/auth/*`, redirect to `/login`
   - If user AND path IS `/login`, redirect to `/`
   - Return response
   
   Export config with matcher: `["/((?!_next/static|_next/image|favicon.ico|.*\\.(?:svg|png|jpg|jpeg|gif|webp)$).*)"]`
  </action>
