**1. Quality Control (`src/features/actions/lib/quality-control.ts`):**

```typescript
export interface QCResult {
  passed: boolean
  reason?: "empty" | "too_many_sentences" | "contains_url" | "mentions_price" | "no_post_reference"
}

export function runQualityControl(dm: string, originalPost: string): QCResult
```

Rules applied in order:
1. Empty check: `dm.trim().length === 0` -> `{ passed: false, reason: "empty" }`
2. Sentence count: split on `/[.!?]+/` filter non-empty, >3 -> `{ passed: false, reason: "too_many_sentences" }`
3. URL check: test against `/https?:\/\/|www\./i` -> `{ passed: false, reason: "contains_url" }`
4. Price check: test against `/\b(price|discount|promo|offer|deal|free trial)\b/i` -> `{ passed: false, reason: "mentions_price" }`
5. Post reference: extract words from originalPost with length > 4, check if any appear in dm (case-insensitive) -> `{ passed: false, reason: "no_post_reference" }`
6. All pass -> `{ passed: true }`

**2. DM Generation (`src/features/actions/lib/dm-generation.ts`):**

```typescript
import Anthropic from "@anthropic-ai/sdk"
import { runQualityControl } from "./quality-control"

export interface DmGenerationInput {
  postContent: string
  productDescription: string
  suggestedAngle: string
}

export interface DmGenerationResult {
  content: string
  passed: boolean
  failureReason?: string
}

export async function generateDM(input: DmGenerationInput): Promise<DmGenerationResult>
```

System prompt (exact text):
```
You are writing a Reddit DM on behalf of a product owner. Rules:
- Max 3 sentences
- No links or URLs
- No mentions of price, discount, or promotion
- Reference something specific from their post
- Casual, helpful tone -- no hard sell
- End with a question or soft CTA
- Do NOT start with "Hey, I saw your post"
```

User message format:
```
Post: ${input.postContent}

Product: ${input.productDescription}

Angle: ${input.suggestedAngle}

Write a DM.
```

On QC failure: retry once with appended system instruction: `IMPORTANT: Your previous attempt was rejected because: ${qcResult.reason}. Fix this issue.`

On second QC failure: return `{ content: "", passed: false, failureReason: qcResult.reason }`

Create the Anthropic client per-call (serverless safety pattern from Phase 2): `const client = new Anthropic()`
