  keywords: string[]

- Accept productDescription, targetCustomer, competitors string[]
- Instantiate Anthropic client per-call (serverless pattern from Phase 2/3)

- Parse response text as JSON (handle markdown code fences in response)
- Return GeneratedKeywords type

- Get authenticated user via createClient
- Upsert product_profiles: name = productDescription, description = productDescription, problem_solved = targetCustomer, competitors = competitors array, keywords = generated keywords, subreddits = generated subreddits
- Insert monitoring_signals rows for each keyword (type: reddit_keyword) and subreddit (type: subreddit)
- Set users.onboarding_completed_at = now()
- Return { success: true }
